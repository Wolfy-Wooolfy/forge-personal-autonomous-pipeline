# سيستم HR

## نظرة عامة

سيستم HR هو نظام إدارة الموارد البشرية متكامل يهدف إلى تلبية احتياجات إدارة الموظفين، الرواتب، الحضور، التوظيف، التدريب، وتقييم الأداء. النظام موجه لموظفي الموارد البشرية، المديرين، والموظفين.

## الميزات الأساسية

- إدارة الموظفين
- الرواتب
- الحضور
- التوظيف
- التدريب
- تقييم الأداء

## المستخدمون

- موظفو الموارد البشرية
- المديرين
- الموظفين

## الافتراضات

- تم تفويض النظام باقتراح كامل للوظائف الأساسية والمستخدمين بناءً على الممارسات الشائعة في أنظمة الموارد البشرية.
- لا توجد متطلبات تكامل أو أمان خاصة محددة في الوقت الحالي.

## السيناريو الموصى به

اقتراح نظام موارد بشرية متكامل يشمل الوظائف الأساسية المذكورة أعلاه، مع دعم المستخدمين الرئيسيين، مع افتراض عدم وجود تكاملات أو متطلبات أمان خاصة ما لم يتم تحديدها لاحقاً.

## حالة الوثائق

تمت الموافقة على الوثائق الرسمية للنظام.

# نظام الموارد البشرية - نموذج أولي

هذا مشروع نموذج أولي بسيط يعرض الميزات والمستخدمين الأساسيين لنظام الموارد البشرية بناءً على المتطلبات المقدمة.

## كيفية التشغيل

1. افتح الملف `index.html` في متصفح الويب الخاص بك.

   - يمكنك فتح الملف مباشرة بالنقر المزدوج عليه أو سحب الملف إلى نافذة المتصفح.

2. ستظهر صفحة تعرض الميزات والمستخدمين للنظام.

## ملاحظات

- هذا النموذج الأولي لا يحتوي على وظائف تفاعلية أو تكاملات.
- الغرض منه هو عرض المتطلبات بشكل مبسط.
- لا يعتبر هذا التطبيق جاهزًا للإنتاج.

## هيكل الملفات

- `index.html`: الصفحة الرئيسية التي تعرض المعلومات.
- `README.md`: تعليمات التشغيل والملاحظات.


---

**تنبيه:** هذا التطبيق هو نموذج أولي فقط ولا يحتوي على أي منطق برمجي أو قاعدة بيانات أو أمان.

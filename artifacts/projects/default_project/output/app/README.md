# HR System Prototype

This is a prototype implementation of an HR Management System backend based on the provided requirements.

## Features Implemented (Prototype)
- Employee Information Management (CRUD limited to create and read)
- Attendance and Leave Tracking
- Payroll Management (basic salary retrieval)
- Performance Evaluation
- Training and Development
- Compliance and Reporting (basic employee listing)

## Assumptions
The system is a prototype and uses an in-memory data store. Data will be lost on server restart.

## How to Run

1. Ensure you have Node.js installed (version 14 or higher recommended).
2. Navigate to this directory in your terminal.
3. Run `npm install` to install dependencies.
4. Run `npm start` to start the server.
5. The server will listen on port 3000 by default.

## How to Use

- The API is RESTful and JSON-based.
- Use tools like Postman or curl to interact with endpoints.
- Example: `GET http://localhost:3000/api/employees` to list employees.

## Note
This is a prototype for demonstration purposes only and not production-ready.

# HR System Prototype

This is a simple prototype web application for the HR System project.

## Features
- Employee Management
- Payroll
- Attendance
- Recruitment
- Training
- Performance Evaluation

## Users
- HR Staff
- Managers
- Regular Employees

## Security
- Privacy compliance
- Data encryption
- Role-based access control

## Integration
- Payroll system
- Attendance system
- Training platforms

## UI
- Multi-language support (Arabic and English)
- User-friendly interface

## How to Run

1. This is a static web application.
2. Open the `index.html` file in any modern web browser.
3. Use the language selector dropdown to switch between Arabic and English.

## Notes
- This is a prototype and does not include backend or real data.
- No actual integration or security features are implemented.

---

### Run Instructions

- Simply open `index.html` in your browser.
- No installation or server setup required.

### Prototype Disclaimer
This application is a prototype for demonstration purposes only and is not production-ready.

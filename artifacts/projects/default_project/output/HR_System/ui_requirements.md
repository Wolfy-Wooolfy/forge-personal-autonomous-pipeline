HR System UI Requirements

1. Multi-language Support
- Support Arabic and English languages.
- Allow users to switch languages seamlessly.

2. User Friendly Interface
- Intuitive navigation for HR staff, managers, and employees.
- Responsive design for desktop and mobile devices.

3. Accessibility
- Ensure compliance with accessibility standards.

4. Features UI
- Employee Management Dashboard.
- Payroll overview and details.
- Attendance tracking interface.
- Recruitment and onboarding modules.
- Training management UI.
- Performance evaluation forms and reports.

5. Security
- Login with role-based access.
- Session timeout and secure authentication.

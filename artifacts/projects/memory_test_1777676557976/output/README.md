مشروع: اعمل سيستم حسابات

الوصف:
نظام حسابات مالي متكامل لشركة متوسطة الحجم يدعم إدارة الحسابات المالية (الدخل، المصروفات، الأصول، الخصوم)، إدارة حسابات العملاء والموردين، توليد تقارير مالية شاملة (قائمة الدخل، الميزانية العمومية، تقارير التدفق النقدي)، دعم تعدد المستخدمين مع صلاحيات مختلفة، تكامل مع أنظمة الرواتب والمخزون، وواجهة نظام ويب للوصول من أي مكان.

المتطلبات الوظيفية:
- إدارة حسابات مالية تشمل الدخل والمصروفات والأصول والخصوم
- إدارة حسابات العملاء والموردين
- توليد تقارير مالية شاملة مثل قائمة الدخل، الميزانية العمومية، وتقارير التدفق النقدي
- دعم تعدد المستخدمين مع صلاحيات مختلفة للتحكم في الوصول
- تكامل مع أنظمة أخرى مثل أنظمة الرواتب والمخزون
- واجهة نظام ويب للوصول من أي مكان

المتطلبات غير الوظيفية:
- أداء عالي مع استجابة سريعة
- أمان عالي لحماية البيانات المالية
- سهولة الاستخدام مع واجهة مستخدم بديهية
- قابلية التوسع لدعم نمو الشركة

الافتراضات:
- النظام موجه لشركة متوسطة الحجم
- التقارير المالية المطلوبة هي التقارير المالية الأساسية المذكورة
- التكامل يشمل أنظمة الرواتب والمخزون فقط
- النظام سيكون نظام ويب

السيناريو الموصى به:
نظام حسابات مالي متكامل لشركة متوسطة يدعم إدارة الحسابات المالية والعملاء والموردين، مع تقارير مالية أساسية، دعم تعدد المستخدمين، تكامل مع أنظمة الرواتب والمخزون، وواجهة ويب.
